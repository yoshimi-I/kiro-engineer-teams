# Kiro Engineer Teams

Template for running 8 Kiro CLI agents in parallel to automate the full development cycle.
8つのKiro CLIを並列で走らせ、開発サイクルを全自動化するテンプレート。

## Getting Started / 使い方

1. Tell me what you want to build (brainstorming starts automatically)
   作りたいものを伝えてください（ブレストが自動で始まります）
2. After design is finalized, issues will be created
   設計が固まったらissueが作成されます
3. Run `./scripts/start-pipeline.sh` to launch the pipeline
   パイプラインを起動します
