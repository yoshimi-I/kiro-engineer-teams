# 質問フォーマット（チャットベース）

## ルール
- チャット会話で直接質問する
- 明確さのために選択式フォーマットを使用
- 関連する質問をグループ化（1メッセージにつき最大3-5問）
- ユーザーの回答を待ってから次に進む
- 回答が曖昧な場合はフォローアップ質問

## フォーマット
```
[質問の背景を1文で説明]

1. [質問文]
   A) [選択肢1]
   B) [選択肢2]
   C) [選択肢3]
   D) その他（教えてください）

2. [質問文]
   ...
```

## ガイドライン
- 日本語で応答
- 質問は簡潔かつ具体的に
- 可能な場合は推奨オプションを提示: 「おすすめ: B」
- 一度に5問以上聞かない
- 次のステージに進む前に回答をサマリー
